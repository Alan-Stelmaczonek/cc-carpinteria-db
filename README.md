# C&C Carpintería – Sistema de Gestión de Base de Datos

Sistema de base de datos relacional desarrollado como proyecto académico para una empresa de carpintería, orientado a la gestión de compras, producción, clientes y ventas.

## Tecnologías Utilizadas
- Microsoft SQL Server
- Git y GitHub
- Metodología Scrum (trabajo por sprints)
- Trello (gestión del proyecto)

## Funcionalidades
- Diseño e implementación de base de datos relacional
- Normalización hasta Tercera Forma Normal (3FN)
- Tablas con claves primarias y foráneas
- Consultas analíticas para métricas de negocio
- Vistas, triggers y procedimientos almacenados
- Reglas de integridad y validaciones de datos
